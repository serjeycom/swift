
## 🚕 Задание: Калькулятор поездки на такси

Вы решили создать приложение для расчёта стоимости поездки на такси. 

Нужно подготовить "движок" расчётов, который будет учитывать разные параметры: 

 - `расстояние`
 - `время подачи`
 - `класс авто`
 - `бонусы пользователя`

----------

### **Часть 1. Переменные, константы и типы данных**

1.  Создайте константу `baseRate` (базовый тариф) типа `Double` со значением `50.0`
    
2.  Создайте переменную `distance` (расстояние в км) типа `Double` и присвойте ей значение `12.5`
    
3.  Создайте переменную `waitingMinutes` (минуты подачи/ожидания) типа `Int` со значением `3`
    
4.  Создайте переменную `driverRating` (рейтинг водителя) типа `Float` и присвойте ей значение `4.8`
    
5.  Создайте переменную `passengerName` (имя пассажира) типа `String` со значением `"Галина"`
    

----------

### **Часть 2. Базовые операторы**

Используя созданные переменные, напишите функцию для расчётов:

6.  **Рассчитайте предварительную стоимость:**
    
    -   Создайте функцию, которая посчитает `стоимость поездки`, по формуле `базовый тариф` × `дистанция`
        
    -   Добавьте функцию, которая посчитает `стоимость ожидания` по формуле: `время ожидания * 5.0` 
    
    -  Сложите базовую `стоимость поездки` и `стоимость ожидания`
        
7.  **Примените повышающий коэффициент за высокий рейтинг водителя:**
    
    -   Создайте функцию, которая повышает стоимость поездки в `1.25` раза, если рейтинг водителя больше `4.5` 
        
    -   _Подсказка:_ используйте тернарный оператор `? :`
        
    -   Пересчитайте `стоимость поездки` с учётом `повышающего коэффициента`
        
8.  **Выведите результат:**
    
    -   Напишите функцию, которая будет выводить сообщение: ` "(passengerName), стоимость поездки: (tripCost) рублей".`
        

----------

### **Часть 3. Опциональные типы**

Теперь представим, что не все данные известны сразу. 

Они приходят с сервера опционально.

9.  **Создайте опционалы:**
    
    -   Объявите переменную `promoCode` (промокод), подумайте какой у неё будет тип данных.
        
    -   Объявите переменную `discountAmount` (размер скидки в процентах) и присвойте ей значение `15`.
        
10.  **Безопасно извлеките опционалы:**
    
        -  Напишите функцию, которая применит скидку, если скидки нет – выведите в консоль `"Скидка не применена"`. 
        - Верните стоимость поездки с учётом скидки
        
11.  **Используйте `guard let` для проверки промокода:**
        -  Напишите функцию, которая принимает в параметр опциональный промокод.
        -  Внутри функции с помощью `guard let` проверьте, равен ли промокод `"SALE50"` если не равен – выведите в консоль `"Промокод не применён"`
        - Верните стоимость поездки с учётом скидки
    
----------

### **Часть 5. Финальный расчёт (всё вместе)**

15.  **Соберите всё воедино:**

        -  Напишите функцию, которая принимает все параметры (расстояние, минуты, рейтинг, промокод, скидку, класс авто) и возвращает финальную цену типа `Double`.
        - Внутри функции последовательно примените все расчёты, которые вы делали в предыдущих частях.
        -  Добавьте проверки входных данных (например, расстояние должно быть больше 0).
