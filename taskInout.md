    Напишите функцию `calculate`, которая принимает `массив целых чисел` и возвращает кортеж: 

	(`Сумма всех элементов в массиве`, `среднее значение` и `наименьший элемент`)

    
3.  **Сортировка массива**  

    Напишите функцию `sortedCopy`, которая принимает массив целых чисел и возвращает **новый** отсортированный массив (по возрастанию). 

	Исходный массив должен остаться неизменным.
    

### Часть 2. С inout (изменение оригинала)

5.  **Обмен значениями**  
    Напишите функцию `swapValues`, которая принимает два целых числа через `inout` и меняет их местами.
    
6.  **Сортировка на месте**  
    Напишите функцию `sortInPlace`, которая принимает массив целых чисел через `inout` и сортирует его по возрастанию. Вызовите её и убедитесь, что **оригинальный массив изменился.**
