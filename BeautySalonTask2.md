
# 💅 Салон красоты — Система управления

### 1.  Расчёт прибыли

Салону необходимо **рассчитать общую выручку**.  
Для этого нужно пройтись по всем мастерам, у каждого мастера есть список записей, а у каждой записи — стоимость услуги.  
Функция должна вернуть **общую сумму** всех оказанных услуг.

---

### 2. Администратор и уборщик

В салоне уже есть **текущий администратор**.  
Теперь нужно добавить **текущего уборщика**.

**Важное условие:**  
Уборщик — **не постоянный сотрудник**. Он может:
- заболеть
- просто не выйти на работу

➡️ В таких случаях **администратор должен иметь возможность подменить уборщика** другим сотрудником (самим собой).

---

### 3. Подтверждение записи по SMS

После того как администратор **назначил запись**, система должна автоматически проставить клиенту, относящемуся к этой записи, поле `isConfirmedBySms = true`.

Это означает, что клиенту было отправлено SMS с кодом, и он успешно его подтвердил.

---

### 4. Перенос записи между мастерами

У администратора должна быть функция, которая **меняет запись между двумя мастерами**.

**Сценарий:**  
Один мастер попросил перенести его запись другому мастеру.

**Что происходит:**
- у первого мастера запись **удаляется**
- у второго мастера эта же запись **добавляется**
