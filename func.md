# Функции в Swift: Полное руководство

Функции — ключевое понятие программирования, это своего рода рабочая единица. 

В программировании очень часто приходится выполнять некую логическую работу – всё это принято оборачивать в функцию.

Функции помогают добиться высокой переиспользуемости кода, а чем меньше кода написано – тем лучше. Идеален тот код, что не написан.

В этой статье мы подробно разберём все аспекты работы с функциями.


## Основы функций

Функция в `Swift` начинается с ключевого слова `func`, за которым следует:

 - `имя функции` 
 - `список параметров в круглых скобках`(если есть)  
 -  `возвращаемый тип` (если есть)

Самая простая функция не принимает параметров и ничего не возвращает:

```swift
func sayHello() {  
    print("Привет, мир!")  
}
```

Отлично – функцию мы написали, но пока её не вызвать, от неё мало смысла. Вызов функции происходит следующим образом:

```swift
sayHello()  // print("Привет, мир!")  
```

Вы не можете вызвать функцию из любого места в коде, вызов функции происходит из другой функции)

Обычно, первая функция, с которой начинается вызов всех остальных – функция `onAppear`, которая вызывается автоматически в момент появления `View` на экране.

Вызов функции может происходить в момент нажатия на кнопку пользователем.

## Параметры функций

Функции могут принимать один или несколько параметров. 

Каждый параметр имеет имя и тип:

```swift
func greet(person: String) {  
	print("Привет, (person)!")  
}

greet(person: "Анна") // Привет, Анна!
```

Несколько параметров разделяются запятыми:

```swift
func calculateSum(a: Int, b: Int) {  
    let result = a + b  
    print("Сумма: (result)")  
}

calculateSum(a: 5, b: 3) // Сумма: 8
```

Для этого мы немного изменим нашу функцию – добавим входные параметры, а так же вернём полученное значение:

```swift
func calculateSum(x: Int, y: Int) -> Int {  
	let result = x + y  
	print("Сумма: (result)")
	return result
}
```

Теперь при вызове функции, мы можем сложить абсолютно любые числа (надеюсь не нужно перечислять какие именно)

```swift
let result = calculateSum(x: 3, y: 5)	 // 8
let result1 = calculateSum(x: 4, y: 9)   // 13
let result2 = calculateSum(x: 10, y: 11) // 21
```

Обратите внимание, что теперь эта функция не просто выводит в консоль (`print("Сумма: (result)")`) она ещё и возвращает `-> Int` посчитанное значение, которое вы можете использовать в последующих расчётах.

## Возвращаемые значения

Функция может возвращать значение, а может и не возвращать.

Когда функция должна возвращать значение ? 

Всё зависит от вашего решения, если вам нужно посчитать сумму между `3` и `5` – скорей всего,  полученный ответ нам понадобится в дальнейшем и мы должны вернуть `Int`

Но что если функция должна просто выполнить какую-то работу и всё ? Например показать `Алерт`, что пользователю ещё нет 18ти лет ?

```swift
func showAlert(user: User) {  
    if user.age < 18 {
        // Показываем Alert
    }
}
```

В таком случае – мы не планируем, что-то из неё возвращать, она просто самостоятельно покажет Алерт. 

> **Заметка**
> 
>  На самом деле – вопрос с возвратом значения из функции станет более чем очевидным, когда вы начнёте писать функции самостоятельно.

Тип возвращаемого значения указывается после стрелки `->:`

```swift
func addNumbers(a: Int, b: Int) -> Int {  
    return a + b  
}

let sum = addNumbers(a: 10, b: 20) // sum = 30
let sum2 = addNumbers(a: sum, b: 30) // sum2 = 60
```

Функция может возвращать несколько значений с помощью кортежа:

```swift
func minMax(array: [Int]) -> (min: Int, max: Int) {  
    // Проверим, что массив не пуст
    if array.isEmpty {
        return (min: 0, max: 0)
    }

    // Так как массив точно не пустой – обращаться по индексу безопасно
    var currentMin = array[0]  
    var currentMax = array[0]

    for value in array {  
        if value < currentMin {  
            currentMin = value  
        } else if value > currentMax {  
            currentMax = value  
        }  
    }

    return (currentMin, currentMax)  
}

let bounds = minMax(array: [8, -6, 2, 109, 3, 71])  
print("Минимум: (bounds.min), максимум: (bounds.max)") // "Минимум: -6, максимум: 109"
```



## Сквозные параметры

По умолчанию параметры функции являются константами. 

Если мы захотим добавить пользователю автомобиль, нам придётся создать копию нашего пользователя, но уже как `var`

```swift
struct Driver {
    // Имя водителя
    let name: String
    // Автомобиль – по умолчанию считаем, что автомобиля нет
    var car: Car?
}

// Водитель Даниель, авто нет
let daniel = Driver(name: "Daniel Morales", car: nil)

// Функция, которая добавляет автомобиль
// И возвращает водителя, но у которого уже есть авто
func addCar(_ car: Car, to driver: Driver) -> Driver {  
    // Придётся создать изменяемую копию водителя
    var driver = driver 
    driver.car = car
    return driver
}
```

Как вы заметили, мы не можем добавить автомобиль водителю напрямую, нам пришлось создать его копию как `var driver = driver`. 

После того как функция вернёт водителя, но уже с автомобилем – нам нужно не забыть, что у нас есть старый водитель (без авто) и новый (который вернула функция) – получается мы должны теперь работать тоьлко с новым водителем, а про старого забыть.

Существует продвинутый способ, как можно этого избежать и поменять значение напрямую:

**Сквозные параметры** (`inout`) позволяют изменять переданные значения прямо внутри параметров  функции:

```swift
func addCar(_ car: Car, to driver: inout Driver) -> Driver {  
    driver.car = car
}

// Водитель Даниель, теперь он должен стать var!
var daniel = Driver(name: "Daniel Morales", car: nil)

// & говорит о том, что значение может быть изменено прямо внутри параметра функции
addCar(Car(model: "Peugeot"), to: &daniel)  
```

Обратите внимание на амперсанд `&` перед аргументами при вызове — он указывает, что переменная может быть изменена прямо в параметре функции, напрямую.

В таком случае – нам не нужно создавать копию водителя, что весьма удобно – теперь у нас только один водитель и мы меняем его напрямую.
