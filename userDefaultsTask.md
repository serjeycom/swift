
## Условие

1. Имеется экран (`View`) со списком мест (например: `Циферблат`, `Гага` и `другие`).

2. При нажатии на кнопку **«Добавить место»** открывается шторка (`.sheet`)

3. Внутри шторки расположено текстовое поле с подсказкой: *«Введите название места»*

4. Также в шторке присутствуют следующие элементы управления:

```swift
Menu {
      Button("Настольные") {  } // Создаём с типом – Настольные
      Button("Кино") {  }       // Создаём с типом – Кино
      Button("Теннис") {  }     // Создаём с типом – Теннис
      Button("Отмена", role: .cancel) {  } // Скрыть меню
  } label: {
     Label("Создать место", systemImage: "plus.circle.fill")
  }
           
   Button("Закрыть шторку", role: .close) {  }
```

5. После выбора типа места:
   - Внутри `ViewModel` создаётся новый объект места
   - Шторка автоматически закрывается

6. Новое место мгновенно отображается в общем списке на экране

7. Все созданные места сохраняются в **UserDefaults**, чтобы они не исчезали после перезапуска приложения.
